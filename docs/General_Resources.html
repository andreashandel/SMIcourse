General Infectious Disease Epidemiology and Modeling Resources

Andreas Handel

2020-06-05 11:08:02

To distinguish clearly which external resources are directly related to the course and which are not, I decided to have two resource pages. The Course Resources page lists materials directly related to and used/mentioned in the course. This page lists a lot of other resources that might not be used or mentioned explicitly in the course, but that might be useful and interesting. Everything listed here is of course related to the course topic, i.e. infectious disease epidemiology (and related areas, e.g. ecology/evolution), with a focus on a model-based perspective. For even more materials, see the links to various lists by others at the end of this document.

Apart from the books, most other materials described below are (should be) freely available online. For better or for worse, a lot of the resources I list below are dynamic and ever changing. That means occasionally links might not work, sites go offline, chapters in online books get re-arranged, etc. If any link does not work and you can’t access the materials for some reason, let me know so I can update this document.

Hands-on/interactive resources

  • Swirl is a package that teaches R inside R. Especially beginner students have found it to be a good start since it provides very encouraging feedback. The downside is that all code writing happens interactively in the console, which is not the way one should write real code. It’s still worth checking out if you want to get some more direct, hands-on R practice.
