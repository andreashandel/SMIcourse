  • Know how to run and use the DSAIRM package.

DSAIRM Introduction

DSAIRM, which stands for Dynamical Systems Approach to Immune Response Modeling is an R package. Its goal is to let users to explore infectious disease models and topics without having to implement their own models and write code. DSAIRM has a dedicated website. Visit the website and read through the Get started document.

DSAIRM hands-on

If you haven’t done already, load DSAIRM. You should see a short message on the screen. Some packages show messages when you load them, and others don’t. In this case, the package tells you that you can start using it by typing dsairmmenu() into the R console. Do that to start it

dsairmmenu()

A menu should open in your browser, from which you can explore different models/apps. You can explore DSAIRM briefly, but the main goal for now is to have it up and running.
