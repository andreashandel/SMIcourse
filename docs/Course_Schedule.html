Simulation Modeling in Immunology - Course Schedule

Andreas Handel

2021-03-31 18:28:03

This is the current course schedule for the class. Things might change, so check regularly for updates.

Notes

Time zones: Seattle is in the PT zone. Your instructors are in ET (Andreas) and CT (Paul) zones. Some of you are likely in yet other locations and time zones. We hope we can keep things straight. For any times in this course we plan to use PT/Seattle time as reference. We also plan to be clear when posting any time by stating the time zone explicitly. If ever in doubt, i.e. if you see us posting a time without specifying the zone, please tell us to clarify.

For details on Slack and Zoom use, see the Communication section.

Abbreviations used: DSAIRM = Dynamical Systems Approach to Immune Response Modeling - R package. Numbers refer to the number of the app.

Before course start

  • Familiarize yourself with this website. Read through all pages apart from the ones under the Modules tab.
  • Go through the materials listed under Module 1. You should end up with R/Rstudio up and running and the DSAIRM package installed and working.
  • Read Simulation Modelling for Immunologists.
